## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/GeoAnalytic-code/daas-catseaice.git
   ```
2. Install requirements
   ```sh
   pip install requirements.txt
   ```   
3. Run tests
   ```sh
   pytest
   ```
